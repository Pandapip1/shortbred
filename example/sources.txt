input_prots.faa - This file contains 75 protein sequences from the Antibiotic Resistance Genes Database (ARDB).

Liu B, Pop M. ARDB-Antibiotic Resistance Genes Database. (2009) Nucleic Acids Res. Jan;37(Database issue):D443-7

ref_prots.faa - This file contains 75 protein sequences from the Integrated Microbial Genomes and Metagenomes database (IMG). http://img.jgi.doe.gov/


wgs.fna - This file contains 100 simulated Illumina reads. They were created using GemSIM, and contains nucleotide sequences from the ARDB and from the Kyoto Encyclopedia of Genes and Genomes.

McElroy et al.: GemSIM: general, error-model based simulator of next-generation sequencing data. BMC Genomics (2012) 13:74.

Kanehisa, M. and Goto, S.; (2000). KEGG: Kyoto Encyclopedia of Genes and Genomes. Nucleic Acids Res. 28, 27-30
